Instructions

Cette application vous permet de capturer, synchroniser et afficher des flux vidéo et audio depuis OBS Studio.

Configuration requise
  1. Ouvrez OBS Studio
  2. Dans le menu Outils, choisissez "WebSockets Server Settings"
  3. Activez le serveur WebSocket et configurez-le avec les mêmes paramètres que ceux définis dans cette application
  4. Configurez une sortie vidéo et audio pour diffuser vers l'adresse localhost:8081 pour la vidéo et localhost:8082 pour l'audio
Utilisation
  • Cliquez sur Démarrer pour commencer la capture du flux
  • Utilisez le curseur de synchronisation pour ajuster le décalage entre l'audio et la vidéo si nécessaire
  • Cliquez sur Arrêter pour terminer la capture
